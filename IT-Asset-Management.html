GOOGLE SHEETS SYNC SETUP
Step 1 of 4

Create a Google Sheet

Your devices will be stored in a Google Sheet that all IT staff can access. It's free and requires no server.

  1. Go to sheets.google.com
  2. Click + Blank to create a new spreadsheet
  3. Name it ITAssetTrack Data
Step 2 of 4

Add the Sync Script

In your Google Sheet, open Extensions → Apps Script, delete any existing code, paste this, then click Save (💾).

Code.gs
function doGet(e) { return handleRequest(e); } function doPost(e) { return handleRequest(e); } // ... (click Copy Code above for full script)
Step 3 of 4

Deploy as Web App

In Apps Script, click Deploy → New deployment. Set these options exactly:

  1. Type: Web app
  2. Execute as: Me
  3. Who has access: Anyone
  4. Click Deploy → authorise → copy the Web app URL
Step 4 of 4

🎉 You're Connected!

ITAssetTrack will now sync all device data to your Google Sheet automatically every 30 seconds. All IT staff using this app with the same Sheet URL will see the same live data.

✅ Devices sync automatically
✅ Manual push/pull available in Settings
✅ Local cache keeps app working offline
✅ Share the Sheet URL with your IT team
